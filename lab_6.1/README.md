Task №6.1
1. Реализовать структуру данных стек (First in last out) на основе указателей или на основе динамического массива. Структура должна быть реализована в 2 файлах: заголовочном файле, который содержит описание классов, и файле исходного кода, который содержит реализацию классов.
В коде можно использовать только библиотеку iostream.

2. Реализованную структуру свернуть в форме библиотеки и подключить к проекту на применение структуры стек. Проект должен содержать функцию проверки правильности скобочного выражения, которая возвращает булево значение. Пользователь вводит набор символов, содержащий (,),[,],{,}. В введенном пользователем выражении проверить правильность расстановки скобок за линейное время (один просмотр выражения). Под правильностью понимается математическая правильность. 

2+4-5(87192479_7348){72648]76823[] -> no

({}) -> yes

({)} -> no

В коде можно использовать только библиотеку iostream и getline из string


Алгоритм решения

Считываем элемент.
Если это (,[ или {, то помещаем ее в стек.
Если это ),] или }, то сравниваем с верхним элементом стека.
Если образовалась пара, то верхний элемент стека удаляем.
Иначе - выражение неправильное.

Когда все элементы считали, то проверяем стек на пустоту.
Если стек пуст, то выражение правильное.
Иначе - выражение неправильное.
