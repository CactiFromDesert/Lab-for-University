Task №4.1
написать набор функций для работы со строками:
проверка на палиндром, поиск подстроки в строке, шифрование, вывести все названия (то, что внутри кавычек) из текста.

В названии файла указать номер группы и фамилию


!!! Разрешено использование только следующих подключаемых файлов: iostream, locale.h, windows.h, cstring

!!! В программе необходимо использовать хотя бы 3 функции из библиотеки cstring. Для всех использованных функций указать, какой тип  данных подается на вход, что возвращает функция и что она делает.

!!! Реализация меню обязательна.

**компиляция Valgrind происходит через "make vallg"(мое вольное сокращение)
